# Translation Service API

Provides translation and localization services for multi-language support.

## Endpoints
- `GET /translate` — Translate text
- `GET /languages` — List supported languages
- `GET /translations/{key}` — Get translation by key
- `POST /translations` — Add a translation
- `PUT /translations/{key}` — Update a translation
- `DELETE /translations/{key}` — Delete a translation

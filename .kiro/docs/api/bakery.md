# Bakery Service API

Domain-specific service for bakery operations (e.g., product catalog, orders, recipes).

## Endpoints
- `GET /bakery/products` — List all bakery products
- `POST /bakery/products` — Create a new product
- `GET /bakery/orders` — List all orders
- `POST /bakery/orders` — Create a new order
- `GET /bakery/recipes` — List all recipes
- `POST /bakery/recipes` — Create a new recipe

## Integrations
- Inventory Service
- Booking Service

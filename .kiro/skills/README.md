# Skills

## Structure

| Path | Content |
| ---- | ------- |
| `develop/` | Development workflow skills (coding, git, cartography, verification, etc.) |
| `rust/` | Rust idioms, patterns, and best practices (~179 skills) |

## Develop Skills

Each skill lives in its own folder with a `SKILL.md` file containing YAML frontmatter (`name`, `description`) and instructions.

| Skill | Purpose |
| ----- | ------- |
| agent-tool-policy | Tool access control for agents |
| cartography | Codebase exploration and mapping |
| coding | Code change workflow (read → TDD → verify → review) |
| context7-docs | Library documentation lookup via Context7 |
| find-skills | Discover available skills |
| git-commit | Conventional commit workflow |
| verification | Evidence-based completion claims |
| writing-skills | TDD-based skill creation |

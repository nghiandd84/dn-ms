# Merchant Service API

Handles merchant onboarding, profile management, and merchant-specific operations.

## Endpoints
- `GET /merchants` — List all merchants
- `POST /merchants` — Register a new merchant
- `GET /merchants/{id}` — Get merchant details
- `PUT /merchants/{id}` — Update merchant profile
- `POST /merchants/onboard` — Onboard a merchant

## Integrations
- Fee Service
- Payment Services
- Inventory Service
